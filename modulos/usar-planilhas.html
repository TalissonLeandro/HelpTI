📊 Como usar planilhas

Aprenda o básico do Excel e Google Planilhas para organizar dados e fazer cálculos simples.

O que é uma planilha?

Uma planilha é uma tabela com linhas e colunas onde você pode organizar informações, fazer cálculos automáticos e criar listas. Os programas mais usados são o Microsoft Excel e o Google Planilhas (gratuito).

Entendendo a estrutura

  • Colunas — identificadas por letras (A, B, C...)
  • Linhas — identificadas por números (1, 2, 3...)
  • Célula — o cruzamento de uma coluna com uma linha (ex: A1, B3)

Como acessar o Google Planilhas (gratuito)

  1. Acesse sheets.google.com no navegador
  2. Faça login com sua conta Gmail
  3. Clique em "Planilha em branco"

Digitando dados

  1. Clique em uma célula (ex: A1)
  2. Digite o conteúdo e pressione Enter para ir para a próxima linha
  3. Pressione Tab para ir para a próxima coluna
💡 Exemplo prático: Na célula A1 escreva "Produto", em B1 escreva "Preço". Depois preencha as linhas abaixo com seus dados.

Fazendo uma soma simples

  1. Clique na célula onde quer o resultado
  2. Digite =SOMA(
  3. Selecione as células que quer somar (ex: B2 até B5)
  4. Feche o parêntese e pressione Enter
  5. Exemplo completo: =SOMA(B2:B5)

Outras fórmulas úteis

  • =MÉDIA(B2:B5) — calcula a média dos valores
  • =MÁXIMO(B2:B5) — mostra o maior valor
  • =MÍNIMO(B2:B5) — mostra o menor valor
  • =CONT.NÚM(B2:B5) — conta quantos números existem
⚠️ Atenção: Toda fórmula começa com o sinal de =. Se esquecer, o programa vai tratar o conteúdo como texto e não vai calcular.

Salvando a planilha

No Google Planilhas, o arquivo é salvo automaticamente. No Excel, use Ctrl + S para salvar.
