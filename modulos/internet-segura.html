🌐 Como usar a internet com segurança

Boas práticas para navegar na internet sem colocar seus dados em risco.

Senhas seguras

Uma senha forte é sua primeira linha de defesa. Uma boa senha deve ter:

  • Pelo menos 8 caracteres
  • Letras maiúsculas e minúsculas
  • Números
  • Símbolos como @, #, !
💡 Dica: Use senhas diferentes para cada site. Nunca use "123456", seu nome ou data de nascimento como senha.

Reconhecendo sites seguros

Antes de digitar dados pessoais em um site, verifique:

  • O endereço começa com https:// (o "s" significa seguro)
  • Aparece um cadeado ao lado do endereço no navegador
  • O endereço do site está correto (ex: bradesco.com.br, não bradescc.com)

Wi-Fi público

Redes Wi-Fi públicas (de shoppings, cafés, etc.) podem ser perigosas.

⚠️ Atenção: Evite acessar internet banking ou digitar senhas em redes Wi-Fi públicas. Use os dados móveis do celular para operações importantes.

Atualizações do sistema

Manter o Windows e os programas atualizados é essencial para a segurança. As atualizações corrigem falhas que podem ser exploradas por hackers.

  1. Clique no botão Iniciar
  2. Vá em Configurações (engrenagem)
  3. Clique em "Windows Update"
  4. Clique em "Verificar atualizações"

Antivírus

O Windows já vem com o Windows Defender, um antivírus gratuito e eficiente. Certifique-se de que ele está ativado.

  1. Clique no botão Iniciar
  2. Pesquise por "Segurança do Windows"
  3. Verifique se todos os itens estão com o ícone verde (ativo)

O que nunca fazer online

  • Nunca compartilhe sua senha com ninguém
  • Nunca clique em pop-ups dizendo que você ganhou um prêmio
  • Nunca baixe programas de sites desconhecidos
  • Nunca envie fotos de documentos para desconhecidos
