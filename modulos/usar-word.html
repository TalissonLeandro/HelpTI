📄 Como usar o Word

Aprenda a criar, editar e salvar documentos de texto no Microsoft Word.

O que é o Word?

O Microsoft Word é um programa para criar documentos de texto, como cartas, currículos, trabalhos escolares e relatórios. Ele faz parte do pacote Microsoft Office.

Como abrir o Word

  1. Clique no botão Iniciar (ícone do Windows no canto inferior esquerdo)
  2. Digite "Word" na barra de pesquisa
  3. Clique em Microsoft Word para abrir

Criando um novo documento

  1. Ao abrir o Word, clique em "Documento em branco"
  2. Uma página branca vai aparecer — é aqui que você vai digitar seu texto
  3. Clique na página e comece a digitar

Formatando o texto

Selecione o texto que quer formatar (clique e arraste o mouse sobre ele) e use a barra de ferramentas:

  • N (Negrito) — deixa o texto em negrito
  • I (Itálico) — deixa o texto inclinado
  • S (Sublinhado) — sublinha o texto
  • Tamanho da fonte — aumenta ou diminui as letras
  • Cor da fonte — muda a cor do texto
💡 Dica: Use Ctrl + N para negrito, Ctrl + I para itálico e Ctrl + S para salvar rapidamente.

Salvando o documento

  1. Clique em Arquivo no menu superior esquerdo
  2. Clique em "Salvar como"
  3. Escolha onde salvar (ex: Área de Trabalho ou Documentos)
  4. Digite um nome para o arquivo
  5. Clique em "Salvar"
⚠️ Atenção: Salve seu documento com frequência enquanto trabalha. Use Ctrl + S para salvar rapidamente e não perder o que digitou.

Imprimindo o documento

  1. Clique em Arquivo
  2. Clique em "Imprimir"
  3. Escolha a impressora e clique em "Imprimir"
